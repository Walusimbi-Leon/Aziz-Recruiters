# Aziz Recruiters Web Application

Welcome to the Aziz Recruiters web application! This project connects workers from Africa to job opportunities in Middle Eastern countries.

## Project Structure

```
aziz-recruiters-web
├── src
│   ├── index.html          # Main entry point of the application
│   ├── about.html          # Information about Aziz Recruiters
│   ├── jobs.html           # Lists job opportunities
│   ├── apply.html          # Application form for job seekers
│   ├── contact.html         # Contact form for inquiries
│   ├── css
│   │   └── styles.css      # Styles for the web application
│   ├── js
│   │   ├── main.js         # Main JavaScript functionality
│   │   ├── api.js          # API functions for job data
│   │   └── validators.js    # Form validation functions
│   └── data
│       └── jobs.json       # Job listings data
├── .gitignore              # Files and directories to ignore in version control
└── README.md               # Project documentation
```

## Features

- **Job Listings**: View available job opportunities tailored for African workers.
- **Application Form**: Submit applications directly through the website.
- **Contact Support**: Reach out for inquiries or assistance.
- **Responsive Design**: The application is designed to work on various devices.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd aziz-recruiters-web
   ```
3. Open the `src/index.html` file in your web browser to view the application.

## Usage Guidelines

- Navigate through the application using the provided links.
- Fill out the application and contact forms as needed.
- Check the jobs page for the latest job opportunities.
